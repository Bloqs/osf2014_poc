Il y a deux fichier pour tester notre projet

2. Test avec le LoadBalancer en utilisant la URL http://localhost/city

Les test importent deux fichier .cvs (testJmeterCitizen.cvs, testJmeterCity.cvs) où il y a les données qu'on va enregistrer dans le DB

Le test sont fait en utilisant 5 iterations avec 10, 50 et 100 threads
